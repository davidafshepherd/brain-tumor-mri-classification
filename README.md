# Brain Tumor MRI Classification
- [brain_tumor_mri_classification.ipynb](brain_tumor_mri_classification.ipynb) contains the fine-tuned ResNet50 model.
- [Kaggle Dataset](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset) is the dataset used to fine-tune and test the ResNet50 model.
#
The model classifies brain tumors using axial-view MRI scans. The types of tumors include:
- Glioma
- Meningioma
- Pituitary
- No Tumor
